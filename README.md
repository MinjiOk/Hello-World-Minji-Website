# Hello World Minji Website

A simple, styled "Hello World" website with a beautiful beach background and personalized greeting.

## Features

- Clean, centered card layout
- Beautiful beach background image
- Purple personalized greeting text
- Fade-in animation on page load
- Responsive design
- Modern CSS styling with shadows and rounded corners

## Technologies Used

- HTML5
- CSS3

## View the Website

Visit the live site at: [https://minjiok.github.io/Hello-World-Minji-Website/](https://minjiok.github.io/Hello-World-Minji-Website/)

## Local Development

Simply open `index.html` in your web browser to view the website locally.
